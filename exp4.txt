Agar pura process EC2/Ubuntu me run karna hai, to sequence ye rahega:

1. Java install

```bash
sudo apt update && sudo apt upgrade -y
sudo apt install openjdk-17-jdk -y
java -version
```

2. Git install

```bash
sudo apt install git -y
git --version
```

3. Repository clone

```bash
git clone <forked-repository-url>
cd simple-java-maven-app
```

4. Maven install

```bash
cd ..
wget https://archive.apache.org/dist/maven/maven-3/3.9.2/binaries/apache-maven-3.9.2-bin.tar.gz
sudo tar -xvf apache-maven-3.9.2-bin.tar.gz -C /opt
sudo ln -sfn /opt/apache-maven-3.9.2 /opt/maven
echo 'export PATH=/opt/maven/bin:$PATH' >> ~/.bashrc
source ~/.bashrc
mvn -version
```

5. Maven build

```bash
cd simple-java-maven-app
mvn clean package
```

6. Maven output run

```bash
java -jar target/*.jar
```

Output:

```text
Hello World!
```

7. Gradle install

```bash
cd ..
sudo apt install unzip -y
wget https://services.gradle.org/distributions/gradle-8.5-bin.zip
sudo mkdir /opt/gradle
sudo unzip -d /opt/gradle gradle-8.5-bin.zip
echo 'export PATH=$PATH:/opt/gradle/gradle-8.5/bin' >> ~/.bashrc
source ~/.bashrc
gradle -v
```

8. Gradle initialize

```bash
cd simple-java-maven-app
gradle init
```

Selections:

```text
yes
2
no
```

9. `build.gradle` edit karke add karo:

```gradle
id 'application'

jar {
    manifest {
        attributes(
            'Main-Class': 'com.mycompany.app.App'
        )
    }
}
```

10. Gradle build

```bash
gradle build
```

11. Gradle output run

```bash
java -jar build/libs/my-app-1.0-SNAPSHOT.jar
```

Output:

```text
Hello World!
```

12. Clean and rebuild

```bash
gradle clean build
```

Important run commands:

**Maven JAR run**

```bash
java -jar target/*.jar
```

**Gradle JAR run**

```bash
java -jar build/libs/my-app-1.0-SNAPSHOT.jar
```
